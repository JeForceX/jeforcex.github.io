Overview

Web Browser Role-Playing Game with Gameplay Elements inspired by Dungeons & Dragons and Storytelling Elements inspired by Fate/Grand Order.

Features
  • Create and customize a Character of your choice with options ranging from Races and Classes to Skills and Ideologies.
  • Experience Colonial Life in the Era of Space Exploration through World Exploration.
  • Contribute to the growth of the Space-faring Colony by fulfilling requests from various sources.
  • Uncover the deepest and darkest secrets of the universe through the official campaign story.
Game Launcher

The Role-Playing Game is designed and optimized for the PC Web Browser.

If you intend to play on a Mobile Device, consider switching to the Landscape Orientation for the best User Experience.

Press the Play Now button to launch the game.

About

JeForceX™ Forge is a GitHub User Website powered by GitHub Pages which functions as a Proof of Concept Demonstration Website as well as a Web Browser Role-Playing Game.

The Forge is coded from scratch and built upon the HyperText Markup Language ("HTML"), Cascading Style Sheet ("CSS"), and JavaScript ("JS") technologies.
